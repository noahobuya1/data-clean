  1,000,000 total records that are matched between two databases. Assume

  600,000 record pairs as matches, and of these 400,000 correspond as
  true matches. The number of TP + FP + TN + FN = 50,000,000.
\end{enumerate}

\begin{enumerate}
\def\labelenumi{\alph{enumi}.}
\item
  (4 points) Given the information above, find the following information
  in the confusion matrix: TP, FP, TN, and FN.

  False positives = Total predicted matches - true positives = 600000 -
  400000 = 200000

  False negatives = Actual positives - true positives = 500000 - 400000
  = 100000

  True negatives = All records - (FP + FN + TP) = 1000000 - 700000 =
  300000
\item
  (1 point) Calculate the accuracy. Comment on the reliability of this
  metric for this problem.

  TP + TN / (TP + TN + FP + FN) = 7/10 = 0.7 (70\%). Accuracy at 70\%
  seems decent but might still be misleading. The classifier could have
  predicted the majority of true negatives or some matches randomly,
  which could result in a relatively high accuracy without actually
  being good at identifying true matches. In imbalanced datasets,
  accuracy may not provide the full picture of classifier performance.
\item
  (1 point) Calculate the precision.

  TP/ TP + FP = 4/6 = 0.667 (66.7\%)
\item
  (1 point) Calculate the recall.

  TP/ (TP + FN) = 4/5 (80\%)
\item
  (1 point) Calculate the f-measure.

  Precision*Recall / (Precision + Recall) = 0.667 * 0.8/ (0.8 + 0.667) =
  0.727
\item
  (1 point) Comment on the reliability of the precision, recall, and
  f-measure for this problem.

  Precision of 66.67\% shows that a fair number of predicted matches are
  false positives, meaning that the classifier is not entirely accurate
  in predicting true matches. Recall of 80\% indicates that the
  classifier is good at capturing most of the actual matches but still
  misses some true matches. F-Measure balances precision and recall,
  giving an overall view of classifier performance. A value of 72.7\%
  shows that the classifier performs moderately well, but improvements
  could be made, especially in reducing false positives.
\end{enumerate}
